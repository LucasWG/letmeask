## Como rodar?

Execute os comandos no seu terminal:

```bash
# Clone o repositório
$ git clone https://github.com/LucasWG94/letmeask.git

# Entre no repositório
$ cd letmeask

# Instale as dependências
$ yarn install

# Execute a aplicação
$ yarn start
```

Acesse em seu navegador http://localhost:3000/ para ver o resultado.
